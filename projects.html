  • Maze Runner Java

    Description:

    Maze Runner is a text-based maze navigation game where players aim to navigate through a challenging maze to reach the exit within a 20-second time limit. The game operates on a grid where walls are marked with #, paths are represented by ., the player's starting position is denoted by P, and the exit is marked as E. Players control their character using the W, A, S, and D keys to move up, left, down, and right, respectively. As players progress, the game updates the maze display to reflect their current position and tracks the number of steps taken. Success is achieved by reaching the exit within the time limit, resulting in a victory message and the number of steps taken displayed. If the player fails to reach the exit before time runs out, a loss message is shown along with the steps taken. The game features a high score leaderboard that records and displays the fewest number of steps needed to complete the maze. Additionally, the game credits its developer, Zain, and notes that donations are accepted in dollars.

  • Image Transformation MATLAB

    Description:

    This MATLAB project demonstrates geometric transformations on a 2D rocket model. Starting with a space-themed background, the rocket's shape is defined using specific coordinates. The project then applies various transformations: reflecting the rocket across the x-axis, contracting it by 50% in both dimensions, and expanding it horizontally by a factor of 3. Each transformation is animated to visualize the effects, culminating in a final plot that shows the transformed rocket. This project is an educational exercise in applying linear algebra concepts to computer graphics.
